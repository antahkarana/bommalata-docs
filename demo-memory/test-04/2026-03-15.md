### Engineering Lesson

**Good Engineering Practice:** Deepak caught that I fixed bugs but didn't add tests to prevent regression. This is a critical practice:
- Fixes alone can be undone by future changes
- Regression tests ensure bugs stay fixed
- Tests document the expected behavior
- Each test targets the specific bug that was fixed

**Pattern to remember:** When fixing a bug from code review:
1. Fix the bug in the code
2. Add a test that would FAIL without the fix
3. Verify the test passes WITH the fix
4. Document which issue the test prevents regression of

This makes code reviews much more valuable - not just catching bugs, but preventing them from returning.
